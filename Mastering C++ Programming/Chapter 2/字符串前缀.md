## 字符串前缀

```c++
auto st1{R"(1234\nabcd)"}; // R代表不对\n等特殊字符进行处理，注意必须加括号
auto st2{LR"(1234\nabcd)"}; // const wchar_t*
auto st3{u8R"(1234\nabcd)"}; // const char*
auto st4{uR"(1234\nabcd)"}; // const char16_t*
auto st5{UR"(1234\nabcd)"}; // const char32_t*

auto st6{LR"(1234\nabcd)"s}; // wstring
auto st7{u8R"(1234\nabcd)"s}; // string
auto st8{uR"(1234\nabcd)"s}; // u16string
auto st9{UR"(1234\nabcd)"s}; // u32string
```

# 初始化一个序列的值

```c++
std::vector<int> v(10);
```

## fill

```c++
std::fill(v.begin(), v.end(), 10);  // 10 10 10 10 10 10 10 10 10 10
```

## fill_n

```c++
std::fill_n(v.begin(), 5, 10);      // 10 10 10 10 10 0 0 0 0 0
```

## generate

fill和fill_n都只能填充单个值，但有时我们希望每次填充的值都是由某个方法生成而来，除了单写一个循环，还可以采用就是generate函数了。

```c++
int i = 1;
std::generate(v.begin(), v.end(), [&i](){return i*i++;}); // 1 4 9 16 25 36 49 64 81 100
```

对应的还有generate_n:

## generate_n

```c++
int i = 1;
std::generate_n(v.begin(), 5, [&i](){return i*i++;});  // 1 4 9 16 25 0 0 0 0 0
```

## iota

iota用于生成等差序列。但这个函数需要引入numeric头文件

```c++
#include <numeric>
```

```c++
std::iota(v.begin()+1, v.end(), 2); // 0 2 3 4 5 6 7 8 9 10
```

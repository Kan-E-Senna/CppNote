# 19. explicit的作用

用在单参数的类构造函数，用来阻止传参时发生的隐式转换。

反例：

```c++
class A{
private:
    size_t N;
public:
    A(size_t a):N{a}{};
    void print(){
        std::cout << N << std::endl;
    }
};

int main() {
    A a = -1;
    a.print();
    return 0;
}
```

```
18446744073709551615
```

加了explicit之后：

```c++
class A{
private:
    size_t N;
public:
    explicit A(size_t a):N{a}{};
    void print(){
        std::cout << N << std::endl;
    }
};


int main() {
    A a = -1;
    a.print();
    return 0;
}
```

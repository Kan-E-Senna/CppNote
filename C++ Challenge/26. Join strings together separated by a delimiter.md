# 26. Join strings together separated by a delimiter

我们有一个字符串数组，希望把这些字符串用一个分隔符拼接起来。

> Input: {"This", "is", "a", "c++", "example"}
>
> output: This is a c++ example.

代码：

```c++
std::string joinString(const std::vector<std::string> &Strs, const char * const separator){
    std::ostringstream os;
    std::copy(Strs.begin(), Strs.end()-1, std::ostream_iterator<std::string>(os, separator));
    os << *(Strs.end()-1);
    return os.str();
}
```

需要注意的点：

line 1: separator的类型是一个const char \* const，char\*代表这是一个原生字符串，第一个const表示指向的内容不可被修改，第二个const表示指针不可被修改。这里不使用string的原因是因为在第三行中的ostream_iterator\<std::string\>的第二个参数要求separator必须是一个const char\* const，如果separator是string，那么就必须要求第三行中的separator进行static_cast\<const char* const\>

line 2: 初始化一个字节流。这里使用ostringstream或者stringstream似乎没有太大区别。

line 3:使用std::copy来进行赋值，std::copy的大致意思如下：

```c++
template<class InputIterator, class OutputIterator>
  OutputIterator copy (InputIterator first, InputIterator last, OutputIterator result) {
  while (first!=last) {
    *result = *first;
    ++result; ++first;
  }
  return result;
}
```

std::copy的作用大致相当于for循环，把first的值赋给迭代器，再让first和result自增，知道first到达last。std::copy经常有如下的用法：

```c++
vector<int> A{1,2,3,4};
vector<int> B(6, 0);
vector<int> C(6, 0);
std::copy(A.begin(), A.end(), B.begin()); // B: 1 2 3 4 0 0
std::copy(A.begin(), A.end(), std::back_inserter(C)); // C: 0 0 0 0 0 0 1 2 3 4
```

**建议：**尽量使用std::copy赋值，避免使用for-loop，std::copy的效率比for-loop稍快一点。

std::ostream_iterator\<std::string\>(os, separator) 会返回一个迭代器，向这个迭代器赋值就相当于向os输入字符串，并且在输入字符串后再添加一个separator的字符串。这里只读到Strs.end()-1，是因为我们需要对最后一个字符串作单独处理，不加seqarator。

line 4：处理最后一个字符串。

line 5：输出。

## 测试

```c++
int main() {
    std::vector<std::string> V{"Hello"s, "C++"s, "Challenge"s, "examples"s};
    auto r = joinString(V, " ");
    std::cout << r << std::endl;
    return 0;
}
```

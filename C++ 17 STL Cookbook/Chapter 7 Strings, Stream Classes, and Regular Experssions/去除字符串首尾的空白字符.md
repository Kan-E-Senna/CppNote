# 去除字符串首尾的空白字符

在数据处理中经常可以看到首尾有空白字符的字符串，可以采用下面的示例代码去除：

```c++
int main() {
    std::string st{"    abcd    "};
    // 核心代码 begin
    const char whitespaces[]{" \r\t\n"};
    auto first = st.find_first_not_of(whitespaces);
    auto last = st.find_last_not_of(whitespaces);
    auto st2 = st.substr(first, last);
    // 核心代码 end
    std::cout << st2 << std::endl;
    return 0;
}
```
